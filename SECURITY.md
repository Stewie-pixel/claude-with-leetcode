# Reporting Security Issues

To report a security issue, please use [Report Security](https://github.com/Stewie-pixel/claude-with-leetcode/issues?q=state%3Aopen%20label%3Asecurity).
We use security-labeled issues for our intake, and do coordination and disclosure here on
GitHub (including using GitHub Security Advisory). The Security Team will
respond within 5 working days of your report issue.
